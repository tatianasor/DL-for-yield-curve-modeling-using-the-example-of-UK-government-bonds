# Применение глубинного обучения для моделирования кривой доходности на примере UK Gilts

Магистерская диссертация · НИУ ВШЭ · 2026
Программа «Финансовые технологии и анализ данных»

**Автор:** Сорокина Татьяна Денисовна

## О работе

Систематическое эмпирическое сравнение восьми моделей прогнозирования кривой доходности государственных облигаций Великобритании (UK Gilts) с применением классических эконометрических бенчмарков, статистических тестов значимости и анализа интерпретируемости извлекаемых факторов.

**Ключевой результат:** классическая факторная модель **PCA + LR** (Test RMSE = 0,01689; R² = 0,9993) статистически значимо превосходит все семь нейросетевых моделей (DM-тест, p < 0,001 для каждой из 28 пар) и является единственной моделью, обыгрывающей наивный бенчмарк Random Walk. Превосходство сохраняется во всех четырёх рыночных режимах тестовой выборки.

## Данные

| Параметр | Значение |
|---|---|
| Источник | Bank of England (spot yields, daily) |
| Период | 02.01.1985 — 30.04.2026 |
| Наблюдений | 15 094 торговых дня |

Тестовая выборка охватывает четыре принципиально различных рыночных режима: COVID-19 (2020–2021), кризис mini-budget (сентябрь 2022), нормализация монетарной политики (2023–2025), стабильный период 2026 года.

## Модели

**Восемь основных моделей:**

| # | Модель | Архитектура |
|---|---|---|
| 1 | PCA + LR | 15 главных компонент → линейная регрессия |
| 2 | AE_dense | 50→128→64→16→64→128→50 (bottleneck 16) |
| 3 | PCA + MLP | 3 фактора → MLP |
| 4 | PCA + LSTM | 3 фактора → LSTM |
| 5 | PCA + GRU | 3 фактора → GRU (lookback = 5) |
| 6 | AE + MLP | 3 топ-латента AE → MLP |
| 7 | AE + LSTM | 3 топ-латента AE → LSTM |
| 8 | AE + GRU | 3 топ-латента AE → GRU (lookback = 5) |

**Три классических бенчмарка:**

- Random Walk
- VAR(1) на 3 PCA-факторах
- Diebold–Li (динамическая модель Нельсона–Сигела с AR(1))

Каждая стохастическая модель обучена с пятью random seeds: {42, 0, 1, 123, 2025}.

## Статистическая проверка значимости

| Процедура | Параметры | Результат |
|---|---|---|
| Diebold–Mariano | 28 уникальных пар моделей | Все p < 0,001; PCA + LR значимо лучше всех |
| Bootstrap CI для R² | 1 000 выборок, перцентильный 95 % | Ширина CI у PCA + LR — 0,0002 (в 14 раз уже всех) |
| Multiple seeds | 5 запусков на каждую нейросеть | Иерархия моделей устойчива; PCA + MLP критически нестабильна |

## Интерпретируемость факторов

**PCA-факторы** (раздел 4.1 ВКР):

| Фактор | Доля дисперсии | Связь с макропоказателем |
|---|---|---|
| PC1 (Level) | ≈ 96 % | BankRate: **+0,89** |
| PC2 (Slope) | ≈ 3,5 % | Spread 10y–2y: **−0,81** |
| PC3 (Curvature) | < 1 % | автономна (все ≤ 0,11) |

**AE-латенты** (раздел 4.1.8 ВКР):

- 2 нейрона из 16 строго мёртвы (dead ReLU)
- 1 нейрон эффективно мёртв (var ≈ 7 × 10⁻⁴)
- Топ-3 латента захватывают только 64 % дисперсии латентного пространства (vs 99,5 % у PCA на исходных данных)
- Корреляция между топ-1 и топ-2 латентами: **+0,97** (критическая избыточность)
